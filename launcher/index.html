主界面

EVE的启动器集成了多种功能，登录、多开、网络测试、切换服务器等等，各种功能可浏览上图。下面详细说一说部分内容。

登陆

点击 LOG IN 开始填写账号信息

启动器可以登陆三种账号：官网（默认）、Steam、Facebook（没见过），后两种账号可在右下角点击对应图标切换登陆（Steam社区被墙，需要科学上网才能登录Steam账号）。忘记密码可在登陆界面点击 Forgot password？ 进行找回。

第一次登录时，会有一个验证

此时你需要到你的的账号对应的邮箱里接收验证码

把你的验证码复制到前面的验证框里然后点 continue 完成登录，此时你的启动器左上角会显示出你刚才登录的账号

账号管理

话不多说，看图

多开

你会发现，启动器添加一个账号后，还可以点击 ADD ACCOUNT 继续添加其他账号。实际上，EVE的启动器提供了一个非常方便的多开功能，在你添加多个账号后，需要启动哪个账号，只需按下账号ID前面的蓝色三角按钮即可。

注意！多开是有严格条件的：

  • 多开的账号必须都是Omega（付费）号！！！Omega（付费）号和Alpha（免费）号是不能同时进入游戏的，理所当然，多开的账号都是Alpha号一样不可以。
  • 如果你通过某手段强制实现Alpha号多开，那么你将面临CCP的降维打击。

设置①

  • Server list 查看各个服务器状态、配置
  • Settings… 设置
  • Shared Cache… 缓存文件相关设置
    • Show in Explorer 在资源文件管理器中打开缓存文件所在的文件夹
    • Move… 移动缓存文件夹
    • Permissions… 如果你的启动器出现文件读取不了之类的错误，可能是启动器没有缓存文件夹的访问权限，修复一下即可
    • Verify 缓存文件出现问题时可以通过这个功能进行检验，会自动修复问题文件，类似于Steam的验证文件完整性
  • Client Settings 客户端设置
  • Start Loglite 一个日志，提供启动详细信息，出问题可以查看

  • Network diagnostics… 网络分析

    注意，这里的各种测试如Basic，测的并非是游戏服务器的延迟。实际上，游戏服务器的ip地址应该是 87.237.34.200
    服务器位于欧洲，所以，国内到服务器的延迟一般都有200ms左右，而众所周知，EVE是个3D聊天室、Excel表游戏，200ms左右的延迟完全可以接受。
    如何知道你的网络玩EVE延迟如何？自行百度 Ping

  • Show in Explorer 在文件夹中显示：
    • Launcher logs 启动器日志
    • Shared cache 缓存文件夹
  • Refresh 刷新启动器
  • About… 关于
  • Exit 退出启动器

完整客户端、离线安装包

EVE默认是微端形式进入游戏，即启动器只下载一些必要的游戏文件，这些文件很小，然后其他资源都是在你玩游戏时一边玩一边下，这样你很快就可以玩了，不用漫长地等待下载完整的游戏客户端。初衷很美好，然而，我国的基本国情摆在这，往往都是在游戏里长时间/无限黑屏，左下角疯狂转圈圈（表示正在下载资源），这就很蛋疼了。

那怎么破？有两种方法：

  • 设置启动器一次性下载好完整客户端，在设置里勾上 Download everything 或者 Dowload the full EVE game client ,然后重启启动器你就会发现启动器下面的更新状态条那里出现一个8G左右的更新，然后你就可以泡杯茶，看个电影，睡个觉，改天再来看看下没下好。（下载速度实在慢得受不了？代理，你值得拥有）

  • 某些地方的网络会出现启动器下载龟速（而你又不考虑挂代理），这时贴心的CCP就给了你一个很好的解决办法：用你喜欢的下载工具（如迅雷）下载离线安装包！

在启动器下载页面可以下载离线安装包,大小10G左右。

启动器：

  • 官网下载
  • QQ群文件

离线安装包：

  • 官网下载
  • 百度云下载（19.1.16上传）

下载离线安装包后，把离线安装包 EVEResources.7z 和启动器exe程序文件放在同一个目录下，然后打开启动器exe安装程序，此时启动器会自动识别出离线安装包一块安装，这样安装完后你只需再稍微更新一下就可以了，不用在启动器里等待下载完整的客户端。详细使用教程见上面百度云共享文件里的docx。

服务器

前置说明：欧服就是世界服，国外仅此一个服务器，因为服务器是在欧洲的，所以大家习惯性说世界服为欧服。

启动器默认就是欧服的正式服 TRANQUILITY （代号宁静），测试服日常对玩家开放的只有 SINGULARITY

  • TRANQUILITY 正式服（默认）以国内时间为准，白天玩家2万左右，下午人数最少，1.7万左右，凌晨会有3万，周末节假日会突破四万。每日固定维护时间为国内时间19：00，一般维护时长10分钟左右（开汉化时提示的1小时更新时长其实是给国服的），有大更新时会长达半小时或者更久。
  • SINGULARITY 测试服，版本号比正式服新，可以体验新活动、新装备、新模式、新舰船等。角色初始数据来自正式服，大概四个月同步一次正式服。账号只可等正式服同步过来才可以在测试服登录，如果你建号在上一次同步之后，那么你无法在测试服登录，慢慢等下一次同步。测试服日常100人左右在线，每天国内时间中午一点日常维护，还有各种不定时、不定时长的维护。关于测试服玩法，详情看贴：测试服玩法指北

如图1右上角，提示 Version info not availiable，Connection error,图2白屏提示找不到文件，这种是典型的网络错误。

如果你出现网络错误，请挂代理/vpn进行科学上网，如果你没用购买代理服务/vpn，不会如何科学上网，请自行谷歌，或者移步看 蓝灯使用 一文免费使用科学上网服务

Error：Could not copy files for tranquility

  • 方法①：启动器右上角/左下角菜单→shared cache→permissions→等待完成→重启启动器

  • 方法②：启动器右上角/左下角菜单→shared cache→verify→等待完成→重启启动器（如果verify按钮灰色点击不了，切换到测试服把测试服更新完）

  • 方法③: 启动器右上角/左下角菜单→shared cache→show in explorer→删除tq文件夹→重启启动器

  • 方法④: 控制面板→卸载程序→eve launcher→垃圾游戏，早日弃坑

游戏启动后卡后台

如果你出现成功启动游戏后，游戏主程序卡后台没弹出，请检查你的缓存文件夹路径是否带有中文。

游戏内经常黑屏，左下角转圈圈

这是因为你没有下载完整客户端，以微端形式进行了游戏，然后资源下载速度缓慢，导致画面不能及时加载出来，所以经常黑屏。左下角的圈圈转表示下载资源中。

游戏黑屏，已下载好完整客户端

  • 如果你的电脑显卡历史久远，请在设置里把 Run clients with DX9 勾上
  • 可能缓存文件出问题，缓存设置里检验一遍缓存文件

steam启动启动器没有账号

通过steam玩EVE，第一次启动时会自动创建一个带 steam:: 前缀的账号ID，如果没有，请多次重启启动器，还是没用，重启steam。

steam启动eve完全没有反应

可能是steam未成功自动安装游戏相关环境，你的电脑缺少相关文件所以打不开。手动定位到你的eve安装目录并双击evelauncher.exe，应该会提示缺少什么，根据提示缺少的内容自行百度下载即可。
